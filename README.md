# T2askDock

T2askDock は、VS Code 内でタスクをローカル SQLite DB で直接管理するための拡張機能です。複数の DB ファイルを同時に扱えるマルチ DB 対応のタスク管理ツールです。

## Features

- **タスクの作成・編集・削除** — タイトル・説明・ステータス・優先度・担当者・期日・タグを管理
- **カテゴリ（プロジェクト）管理** — タスクをカテゴリに分類して整理
- **サブタスク** — タスクを階層構造で管理
- **カンバンボード** — ドラッグ＆ドロップでステータスを更新
- **リストビュー** — テーブル形式でタスクを一覧表示・ソート
- **DB テーブルビュー** — DB 内の全カテゴリとタスクをツリー形式で表示
- **マルチ DB 対応** — 複数の SQLite ファイルを同時にマウントし、それぞれ独立して操作
- **読み取り専用モード** — DB への誤書き込みを防ぐロックモード
- **AI タスク作成** — AI を使ったタスク生成

## Installation

### VSIX からインストール

1. `.vsix` ファイルをダウンロードします。
2. VS Code を開きます。
3. コマンドパレットから `Extensions: Install from VSIX...` を実行します。
4. ダウンロードした `.vsix` ファイルを選択します。

## Usage

### サイドバー

アクティビティバーの Task Dock アイコンをクリックすると、2 つのツリービューが表示されます。

- **My Recent Tasks** — 自分に割り当てられた直近のタスク
- **All Projects** — 全カテゴリとタスクの一覧。DB ノード・カテゴリノード・タスクノードの階層構造

### ビュー

| ビュー | 開き方 | 説明 |
| --- | --- | --- |
| カンバンボード | カテゴリをクリック | Todo / In Progress / Blocked / Done のカラム表示。ドラッグ＆ドロップでステータス変更 |
| DB テーブルビュー | DB ノードをクリック | DB 内の全カテゴリ・タスクをツリー表示。カテゴリをシングルクリックで開閉、ダブルクリックでボードへ遷移 |
| タスク詳細 | タスクをクリック | タスクの詳細情報の確認と編集 |

### タスク操作

- ツリービューのカテゴリを右クリック → **このカテゴリにタスクを作成**
- DB ノードを右クリック → **このDBにタスクを作成**
- タスクを右クリック → **サブタスク作成 / 編集 / 削除**

### DB 管理

ステータスバー左下の DB 名をクリックすると DB 選択メニューが開きます。

| 操作 | 説明 |
| --- | --- |
| DBファイルをマウント | 既存の `.sqlite3` ファイルをマウント |
| DBフォルダを登録 | フォルダ内の全 `.sqlite3` ファイルを自動検出 |
| 新しいDBを作成 | 新規 SQLite ファイルを作成 |
| Toggle ReadOnly | 読み取り専用モードの切り替え |

## Commands

| Command | Description |
| --- | --- |
| `Task Dock: Open Board` | カンバンボードを開く |
| `Task Dock: Open Table` | テーブルビューを開く |
| `Task Dock: Select Database` | DB 切り替えメニューを開く |
| `Task Dock: Toggle ReadOnly` | 読み取り専用モードの切り替え |
| `Task Dock: Create Task` | タスクを作成 |
| `Task Dock: このDBのリストを表示` | DB テーブルビューを開く |
| `Task Dock: このDBにタスクを作成` | 指定 DB にタスクを作成 |
| `Task Dock: このカテゴリにタスクを作成` | 指定カテゴリにタスクを作成 |
| `Task Dock: DBファイルをマウント` | SQLite ファイルをマウント |
| `Task Dock: DBフォルダを登録` | フォルダを DB ソースとして登録 |
| `Task Dock: 新しいDBを作成` | 新規 DB を作成 |
